### Getting Started

- Compile the project

- Run the command

```
docker-compose up --build --force-recreate
```

- API access at in

``` bash
http://localhost:8080
```
- For debugging

``` bash
http://localhost:5005
```
- Run tests

``` bash
mvn test
```
